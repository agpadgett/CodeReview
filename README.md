# _Word Puzzle App_

##### _Replaces Vowels in words with '-', 08/14/15_

#### By _Andrea Padgett_

## Description

This program replaces vowels in a word or sentence with "-". It assumes that if y's at the end of a word, y is acting as a vowel. 

## Setup

Enter a sentence or word when prompted, hit 'Play'

## Technologies Used

Java
